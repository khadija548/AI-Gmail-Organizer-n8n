# AI Gmail Organizer & Academic Assignment Tracker

## Overview

This project is an AI-powered Gmail automation workflow built using **n8n**, **Google Gemini AI**, **Gmail**, and **Google Sheets**.

It automatically classifies incoming emails, creates Gmail labels, generates smart draft replies, extracts academic assignment information, and stores assignment details in Google Sheets.

---

## Features

- AI-powered email classification
- Automatic Gmail label creation
- Automatic Gmail label assignment
- Email priority detection
- Smart Gmail draft generation
- Assignment detection
- Assignment detail extraction
- Google Sheets integration
- Structured JSON AI output
- Modular workflow design

---

## Workflow

1. Gmail Trigger monitors incoming emails.
2. Gemini AI analyzes email content.
3. AI classifies the email.
4. Gmail labels are created automatically.
5. Labels are applied.
6. AI determines whether a reply is required.
7. Gmail draft is created.
8. Assignment details are extracted.
9. Data is stored in Google Sheets.
